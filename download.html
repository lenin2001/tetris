Tetris - Классическая игра

Это классическая игра Tetris, написанная на HTML, CSS и JavaScript.

Скачать проект

Инструкции по загрузке на GitHub:

  1. Скачайте архив проекта, нажав на кнопку выше
  2. Распакуйте архив в удобное место на вашем компьютере
  3. Создайте новый репозиторий на GitHub с именем "tetris"
  4. Загрузите файлы в репозиторий, используя следующие команды:
cd путь/к/распакованному/архиву
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/lenin2001/tetris.git
git push -u origin main

Настройка GitHub Pages:

  1. Перейдите в настройки вашего репозитория на GitHub (Settings)
  2. Прокрутите вниз до раздела "GitHub Pages"
  3. В разделе "Source" выберите ветку "main" и папку "/ (root)"
  4. Нажмите "Save"
  5. Через несколько минут ваша игра будет доступна по адресу:
    https://lenin2001.github.io/tetris/

Если у вас возникли вопросы или проблемы, обратитесь к документации GitHub или создайте issue в репозитории.
